Task 8.90E+14

Example

Input Image Explanation

  • In the input, you should see...two pixels of different colors on the upper or lower rim, or on the right or left rim.

MC-LARC options

To make the output, you have to...first, copy the input grid and leave it as it is. Then, count the black color grid inside the shape. Next, find another area in the large grid with the same number of black grid as in the first shape, however, it will only be black grid without the shape and the black grid might also be surrounded by black squares or have additional black squares around it. After you find the black grid, replicate the first shape with the black grid you find as the center. There should be two identical shapes in the output grid.

Option 1

To make the output, you have to...first, convert all other colored grids into black. Then, pick the first black grid you see as the center of a new shape. However, the new shape will have more black grids than the original and won't be an exact replica.

Option 2

To make the output, you have to...first, redraw the input grid in reverse order. After flipping the grid, count the number of blue squares within the shape. Copy the shape into a different part of the grid, ensuring that the new shape has fewer black squares.

Option 3

To make the output, you have to...first, remove all colors from the input grid and replace them with black. Then, select a random point in the grid and use it as the center to draw a new shape. Both shapes should have different black grid counts.

Option 4

To make the output, you have to...first, replace the input grid with an entirely black grid. Then, anywhere in the grid, create a new shape using the count of green squares in the original shape. The end result will not have two shapes with the same black grid count.
